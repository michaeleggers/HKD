# Requirements Windows

- Visual Studio 2022
- CMake
- OpenGL 4.6

# Requirements Linux

- Clang(++)
- Cmake
- make
- libXext-devel
- mesa-libGL-devel
- OpenGL 4.6

# MacOS

No support for MacOS. OpenGL only supported up to version 4.1.
And this version did not run stable.
Native Metal backend would be nice in the future.
